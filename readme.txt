cmd창을 실행했을 때 글자색이 제대로 출력되지 않는 경우 해결법(윈도우10 기준)
	: 윈도우버튼 + R 로 ‘실행’ → ‘regedit’ 입력해서 레지스트리 편집기 실행 → HKEY_CURRENT_USER→Console 에 들어와서 우클릭
	 → 새로만들기 → DWORD로 새 값을 생성 →VirtualTerminalLevel로 이름 바꾼 뒤 값을 1로 설정
커스텀 주사위(두 개의 주사위 눈 input으로 입력가능) 사용법
	: 주사위를 굴려야 할 때 cr, 커, croll, ㅋㅈ 중 하나를 입력하면 값을 입력한대로 주사위가 굴러가는 커스텀 주사위를 사용할 수 있다.
전 재산을 같이 보고 싶은 경우 cmd창의 배경을 밝은색으로 바꾸면 검정글씨로 전 재산도 출력됩니다.
